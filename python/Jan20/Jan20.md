### 느낌

또한 변수 선언 위치에 따라서 결과가 달라질 수 있기에 적절한 위치에서 변수를 선언할 필요가 있는 것 같다.

error exception의 경우, 집중력이 떨어져 복습하고 강의를 다시 들어보면 좋을 것 같다.
